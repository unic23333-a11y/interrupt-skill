# 插一句嘴

> 让AI听懂你的"插嘴"，智能判断是修改当前任务还是开新任务。

---

## 这是什么

在使用AI助手时，你是否遇到过这些情况：

- AI正在生成，你突然想补充或修改
- AI跑偏了，你不想等它生成完再纠正
- 你想做新任务，但又不想丢失当前的进度

「插一句嘴」就是为了解决这些问题的Skill。

---

## 核心功能

### 1. 智能判断

AI会自动判断你说的话：
- **和当前任务相关** → 修改/增补当前任务
- **和当前任务无关** → 直接回答，不打断

### 2. 修改原任务

在AI生成过程中，你说的话会被智能解析：

| 你说 | AI理解 | 处理方式 |
|------|--------|---------|
| "，把简历里加上Python" | 修改指令 | 直接修改，继续生成 |
| "，再加一个作品集部分" | 增补指令 | 暂停，增补，继续生成 |
| "，不对，应该是..." | 纠正指令 | 暂停，更新上下文，重新生成 |
| "，先帮我查个东西" | 新任务 | 后台挂起当前，开始新任务 |

### 3. 多任务队列

最多同时管理3个任务：

```
队列满了？→ 提示先完成一个
队列没满？→ 后台挂起当前，开启新任务
```

---

## 使用方法

### 场景A：AI正在生成内容时

直接输入你想说的话，AI会自动判断：

```
AI：正在分析这份数据报告...
你：先把结论部分提前
AI：[理解这是修改指令] → 调整结构 → 继续分析

---

AI：正在写一份方案
你：等等，先确认一下目标用户是谁
AI：[理解这是暂停确认] → 停下来等你说
```

### 场景B：AI没有在生成时

以**逗号开头**（英文`,`或中文`，`）激活打断模式：

```
你：，帮我查下明天的天气
AI：[后台挂起当前任务] → 开始查天气

---

你：，有个紧急的事要处理
AI：[后台挂起] → 处理新任务
```

---

## 优先级说明

这些词出现时，会被视为**新开任务指令**：

| 关键词 | 含义 |
|--------|------|
| 先 | 先做这个，暂停当前的 |
| 后 | 等当前做完再做这个 |
| 等等 | 暂停等一下 |
| 别 | 停止当前任务 |
| 停一下 | 暂停当前任务 |
| 算了 | 取消当前任务 |

---

## 注意事项

1. **队列上限3个** — 队列满时会提示先完成一个
2. **修改后会重新生成** — 不是原地编辑，是重新生成完整内容
3. **上下文会保留** — 切换任务时不会丢失进度

---

## 解决的问题

| 问题 | 解决方式 |
|------|---------|
| AI跑偏了还要等生成完 | 生成中随时可打断/纠正 |
| 新任务怕打断旧任务 | 后台挂起，任务不丢 |
| 上下文被污染 | 任务隔离，互不干扰 |
